# Требования для запуска
- Go 1.23
- Docker

# Инструкция по установке и запуску
1. Склонировать репозиторий:

`git clone https://github.com/XaxatonInoHack/backend.git`

2. Запустить сервис в докере

`docker-compose up --build -d`
